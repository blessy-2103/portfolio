# Blessy Christrus D - Portfolio

A simple, responsive, and modern personal portfolio website built as a single-page application. It showcases my skills, education, internship experience, and projects in Full-Stack Development, AI, and Cybersecurity.

## 🚀 Live Demo
You can view the live website here: https://blessychristrus-portfolio.netlify.app/

## ✨ Features
- **All-in-One File:** Clean layout contained entirely within a single `index.html` for easy management.
- **Dark/Light Mode:** Includes an interactive theme toggle switch.
- **Project Filter:** Dynamic category filtering (All, AI, Security, Full-Stack) built with native JavaScript.
- **Fully Responsive:** Completely optimized for viewing on mobile devices, tablets, and desktops.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- Font Awesome Icons

## 📂 Getting Started
1. Clone or download this repository.
2. Open the `index.html` file directly in any modern web browser to view it locally.
